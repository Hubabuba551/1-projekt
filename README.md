# 💣 Saper

Klasyczny Saper zbudowany w czystym HTML/CSS/JS — bez żadnych zależności.

## Jak grać

- **Kliknij** pole, żeby je odkryć
- **Prawy przycisk myszy** (desktop) lub **długie przyciśnięcie** (mobile) → flaga 🚩
- Przycisk **🚩 Tryb flagi** — zamienia kliknięcie w flagowanie (wygodne na mobile)
- Odkryj wszystkie pola bez min → wygrywasz

## Poziomy

| Poziom  | Plansza  | Miny |
|---------|----------|------|
| Łatwy   | 9 × 9    | 10   |
| Średni  | 14 × 14  | 30   |
| Trudny  | 16 × 16  | 50   |

## Uruchomienie

Otwórz `index.html` w przeglądarce — gotowe.

## GitHub Pages

1. Wrzuć repo na GitHub
2. Settings → Pages → Branch: `main`, folder: `/ (root)`
3. Gotowe pod `https://<twój-nick>.github.io/<nazwa-repo>`.
